# 02135 - Assignment 3 Getting started with the Huzzah32 board and with MicroPython

## Instructions for running the different tasks

To run the different tasks simply change the file to be executed in main.py. So execfile("/Task1/task1.py") for task 1, execfile("/Task2/task2.py") for task 2 and so on.

## Components

| Component | Pin(s) |
|-----------|--------|
| Orange LED / Blue LED | 12 |
| Green LED | 15 |
| Red LED | 33 | Digital Output |
| Button | 27 | Digital Input |
| Temperature Sensor | 21, 22 |
| Potentiometer | 14 |
